# 事件系统对比 - 2: 错误事件

## Agora RTC 错误事件

```typescript
// 错误事件监听
client.on("error", (error) => {
  console.error("Agora RTC 错误:", error);

  switch (error.code) {
    case "OPERATION_ABORTED":
      console.log("操作被中止");
      break;
    case "INVALID_ARGUMENT":
      console.log("参数无效");
      break;
    case "NOT_SUPPORTED":
      console.log("不支持的操作");
      break;
    default:
      console.log("未知错误");
  }
});
```

## 阿里云 RTC 错误事件

```typescript
// 错误事件监听
client.on("error", (error) => {
  console.error("阿里云 RTC 错误:", error);

  switch (error.code) {
    case "OPERATION_CANCELLED":
      console.log("操作被取消");
      break;
    case "INVALID_PARAMETER":
      console.log("参数无效");
      break;
    case "NOT_SUPPORTED":
      console.log("不支持的操作");
      break;
    default:
      console.log("未知错误");
  }
});
```

## 错误对象类型定义

**Agora RTC**:

```typescript
interface AgoraError {
  code: string;
  message: string;
  data?: any;
}
```

**阿里云 RTC**:

```typescript
interface AliError {
  code: string;
  message: string;
  data?: any;
}
```
